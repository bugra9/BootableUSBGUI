### Kaynak koddan yükleme
```
git clone https://github.com/bugra9/BootableUSBGUI
cd BootableUSBGUI
yarn install
```

### Kaynak koddan çalıştırma
```
yarn start
```

### Paketleme
```
yarn dist
```
